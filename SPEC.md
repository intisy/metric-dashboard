# Metric Dashboard - Specifications & Test Requirements

## Goal
Comprehensive usage history and billing dashboard.

## Requirements
- [ ] **History Tracking**: Accurately displays the full history of usage across both OpenCode and Claude Code.
- [ ] **Launch Behavior**: Mimics the plugin-updater logic; runs either on Hub launch or standard application launch.
- [ ] **Concurrency Safe**: Capable of running seamlessly without data corruption or crashes even when multiple OC/CC instances are open simultaneously.
